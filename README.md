# ICT-Governance-and-Policy-Analysis
IPP221 Assessment 2 - ICT Governance and Policy Analysis

A professionally formatted academic report analysing the ICT governance strategy of United Parcel Service (UPS), submitted as part of IPP221 IT Professional Practice at Torrens University Australia.

The report critically evaluates UPS's multi-layered IT governance framework against established international standards including COBIT, ITIL, ISO 27001, COSO, and ISO 9000. It examines the alignment between business strategy and IT investment, assesses the ethical dimensions of centralised data infrastructure, and provides targeted recommendations grounded in academic literature.

Document features:
- Professional title page and table of contents
- Two structured comparison tables with styled headers
- Running headers and footers with page numbers
- APA 7th edition references

Subject: IPP221 IT Professional Practice
Institution: Torrens University Australia
